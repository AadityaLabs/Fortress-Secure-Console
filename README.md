Fortress Secure Console is an advanced security platform designed to protect sensitive data and system access. It offers secure authentication, encrypted communications, access control, and real-time monitoring, helping individuals and organizations maintain a safe, reliable, and protected digital environment.

# Fortress8D Secure Console

Fortress8D Secure Console is a browser-based password generator and vault interface with an immersive cyber-security UI. The project includes an 8D animated canvas background, entropy-based password strength scoring, secure key storage, and a password audit utility.

## Features

- Random password generator with character and passphrase modes
- Adjustable entropy length and word count
- Options for uppercase, lowercase, numbers, symbols, and ambiguous character exclusion
- Passphrase builder with separator and capitalization controls
- Real-time entropy calculation and crack-time estimation
- Save passwords to a local vault with category tags
- Vault search, delete, and export functionality
- Password audit panel with length, complexity, dictionary, and repeat checks
- Copy to clipboard and secure key preview toggle
- Animated 8D particle background and futuristic UI effects

## Built With

- HTML
- Tailwind CSS (CDN)
- JavaScript
- Lucide icons
- Google Fonts

## Project Structure

- `index.html` — main interface and layout
- `style.css` — custom styling and visual effects
- `app.js` — password generation, vault, UI controls, and canvas animation

## Usage

1. Open `index.html` in a browser.
2. Use the Generator tab to create a secure password or pronounceable passphrase.
3. Customize settings for entropy length, character sets, separators, and exclusions.
4. Copy generated keys to the clipboard or save them to the local vault.
5. View saved entries in the Vault tab and manage them with peek, copy, and delete actions.
6. Use the Strength Auditor to analyze any password for risk factors.

## Notes

- The app runs entirely in the browser with no installation required.
- Saved vault items are stored locally in `localStorage`.
- This project is ideal for portfolios, security demos, or UI showcases.
